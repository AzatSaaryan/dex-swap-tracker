## Установка

1. Клонируем репозиторий:
   git clone https://github.com/AzatSaaryan/dex-swap-tracker.git
2. Устанавливаем зависимости:
   npm install
3. Создаем `.env` и добавляем:
   RPC_URL=https://mainnet.infura.io/v3/
   ALCHEMY_KEY=
   NFT_CONTRACT=
   TOKEN_API_KEY=
4. Запуск:
   npm run start

## Пример

```bash
node index.js
```
